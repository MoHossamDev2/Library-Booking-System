# Library Booking System

A modern full-stack Library Booking System built with React, Spring Boot, and MySQL.  
This system allows users to browse and reserve books online, while administrators can manage books and users efficiently.

---

## Technologies Used

- Frontend: React.js (with Hooks & Context API)  
- Backend: Java Spring Boot  
- Database: MySQL  
- Containerization: Docker & Docker Compose  
- Authentication & Security: JWT & Spring Security (optional)  
- Version Control: Git & GitHub  

---

## Features

User Features:  
- Register and login  
- Browse available books  
- Reserve and cancel reservations  
- View reservation history  

Admin Features:  
- Add, update, and remove books  
- Manage users and reservations  
- Dashboard
